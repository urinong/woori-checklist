■ 물품사양서 기재 점검
──────────────────────────────────────

【미기재 항목】
1. 원재료 배합비(%): 7개 원재료 중 5개(현미유, 분당, 바나나농축분말, 알파미분, 천일염)의 
   배합비가 공란 → 규정상 원재료명 및 함량은 전 성분 표기 필수 (표시사항 제6항)
2. 설비위생(항목20): 완전 공란 → 기재 필요
3. 영양성분표시(항목16): "제품 별도 표기"로 기재. 과자(유탕처리제품)는 영양성분 표시 
   의무 대상 (운영규정 제6호) → 실제 제품 포장에 표시 여부 확인 필요

【오류 또는 확인필요 사항】
1. 물품의 특징에 "무첨가", "기름에 튀기지 않고 가볍게 구어내어"라고 기재했으나, 
   식품유형은 "과자(유탕처리제품)"으로 되어 있음 → 유탕처리제품은 기름에 튀긴 것을 의미. 
   물품 설명과 식품유형 간 모순 가능성 → 제조원에 확인 필요
2. "팽창제(탄산칼슘)을 과감히 제외"라고 기재 → 탄산칼슘은 유기가공인증 허용 첨가물 
   (운영규정 제4호 25번)이므로 사용해도 규정 위반은 아님. 참고사항.
3. 원재료 중 "대두" 알레르기 표시가 있으나, 원재료표에 대두 또는 대두 유래 원료가 
   보이지 않음 → 제조 시설 공유에 의한 교차오염 표시인지, 또는 원재료표에 누락된 
   대두 성분이 있는지 확인 필요

【확인 필요 사항 — 위원회 논의 권장】
1. ⚠ 옥수수그릿츠(45%) Non-GMO 확인

   - 국산 옥수수이나, 국내에서도 수입 옥수수를 가공하는 경우가 있으므로 
     제조원(한국곡산)에 Non-GMO 확인서 또는 IP(Identity Preserved) 증명 요청 필요

2. ⚠ 바나나농축분말의 수입원료 허용 여부
   - 근거: 취급기준(원부재료) 제3항 "국내 미생산/확보 곤란 원료의 예외적 허용"
   - 바나나는 국내 미생산 과일이므로 예외 허용 논리 적용 가능하나,
     운영규정 제1호 허용 수입원료 목록에 명시되어 있지 않음
   - 유기원료 여부도 미확인 → 위원회에서 허용 여부 판단 필요

3. ⚠ 현미유 태국산 — 원산지 확인
   - 근거: 운영규정 제1호 기름류 "현미유"는 허용 수입기름에 포함
   - Non-GMO 작물 유래이므로 GMO 기준은 통과
   - 원산지증명서 제출 필요

4. ⚠ 분당(icing sugar) 원당 원산지 및 등급
   - "국내가공"이나 원당의 원산지(수입 원당 가공 가능성)와 유기농 여부 확인 필요
   - 당류기준: "백설탕 이상" → 분당이 백설탕 가공품이면 적합

5. ⚠ 회원원재료사용 "X" — 정회원 원료 미사용
   - 협력회원이 회원원재료를 사용하지 않는 물품
   - 취급기준(공통원칙) 제2항 "가농회원 생산 원료 우선"에 해당하지 않음
   - 규정 위반은 아니나, 심의 시 고려 사항
