■ 물품사양서 기재 점검
──────────────────────────────────────

【미기재 항목】
1. 품목보고번호(항목11): 물품사양서 본문에는 공란이나, 
   별도 제출된 품목제조보고서에 20120275175122 기재 확인
2. 생산관리(항목17): 공란 → "가톨릭농민회" 또는 제조원 기재 필요
   (표시사항 제10항: 생산관리 = 가톨릭농민회)
3. 공급가 불일치: 상단 표 "4,000원" vs 하단(2018년 개정 양식 기준) "2,750원"
   → 현재 적용 가격 확인 필요

【오류 또는 확인필요 사항】
1. 물품사양서 양식이 "2018년 제2차 우리농 전국물품위원회 개정"으로 되어 있음
   → 최신 양식(2026년 개정)으로 갱신 권장
2. 알레르기물질 "해당사항 없음" 기재
   → 가자미는 21가지 알레르기 유발물질 목록에 직접 포함되지 않으므로 적합.
   단, 제조시설에서 알레르기 유발 수산물(새우, 게, 고등어, 홍합, 전복, 굴, 
   조개류, 오징어 등)을 함께 취급하는지 교차오염 가능성 확인 권장.
3. 영양성분표시 "제품 별도 표기" 
   → 기타수산물가공품은 영양성분 표시 의무대상이 아닐 수 있으나(운영규정 제6호 
   수산가공분과 참조), 참고용 시험성적서에 영양성분 데이터가 제출되어 있어 
   실제 표시 가능 상태.


■ 규정 적합성 점검
──────────────────────────────────────

【위반 사항】
없음

【확인 필요 사항】
1. ⚠ 가자미 원산지 세부 확인
   - 수산분과 원재료기준 제1항: "국내 연근해산만 사용 가능"
   - 수매확인서(p.10)에서 ㈜동양엠푸드가 경남 통영 소재 통영수산업협동조합을 
     통해 수매한 것으로 확인 → 국내 연근해산 기준 충족 가능성 높음
   - 다만 수매확인서의 수매 내역에 "병어 7, 가자미 298, 삼치 2"로 기재되어 
     있어 가자미 전용이 아닌 혼합 수매 → 가자미의 구체적 어획 해역 확인 권장

2. ⚠ 수산분과 생산기준 제2항 적용
   - "항생물질 또는 화학첨가물을 사용한 수산물은 취급하지 않는다"
   - 원재료가 가자미(99.8%) + 천일염 + 녹차로 매우 단순하여 화학첨가물 미사용 확인
   - 양식 가자미인 경우 항생물질 사용 여부 확인 필요 (자연산/양식 구분 미기재)

3. ⚠ HACCP 인증서 유효기간
   - 제출된 HACCP 인증서 3건의 유효기간: 2023.06.09 ~ 2026.06.08
   - 현재(2026.04) 유효하나, 약 2개월 후 만료 → 갱신 여부 확인 필요

4. ⚠ 생산관리 항목 공란
   - 표시사항 기준상 "생산관리: 가톨릭농민회"를 기재해야 함
   - 미기재 상태 → 재검토 사유에 해당
